# Saucedemo Login Tests

Automatiserade tester för inloggningsfunktionen på [saucedemo.com](https://www.saucedemo.com) med Selenium WebDriver, Java och JUnit 5.

## Testfall

- Lyckad inloggning med korrekta uppgifter
- Fel användarnamn – felmeddelande visas
- Fel lösenord – felmeddelande visas

## Krav

- Java 11 eller högre
- Maven
- Google Chrome

## Kör testerna

```bash
git clone https://github.com/TurgayEk/saucedemo_test_turgayek.git
cd saucedemo_test_turgayek
mvn test
```
